# Decision Debt DAWO26 Reproducibility Package

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20406810.svg)](https://doi.org/10.5281/zenodo.20406810)

This repository contains the focused reproducibility artifact for the DAWO26 paper:

> **Decision debt: a framework for reasoning about avoidable information loss in collective decision procedures for DAO governance**  
> Deyan Paroushev, Sofia University St. Kliment Ohridski / Advisa EOOD

The package contains the capacity matrix, model implementation, robustness/sensitivity analysis, and regression tests used for Section 5.2 and Table 2 of the paper.

## What this repository is

This is a reproducibility and sensitivity-analysis package for a **conditional decision-support model**. It allows other researchers to:

- inspect the twelve-method by nine-dimension capacity matrix;
- reproduce the worked-example issuance values;
- reproduce Table 2 frontier shares;
- inspect dominance relations and redundant capacity profiles;
- rerun scalarisation checks under L1, L2, and max norms;
- rerun adequacy-function sensitivity checks;
- inspect per-entry capacity sensitivity;
- rerun dimension-ablation and weight-space stability checks;
- modify the matrix or decision profiles and observe how frontier sets change.

## What this repository is not

This repository **does not validate** the Decision Debt framework empirically. In particular, it does not establish that:

- the capacity matrix is objectively correct;
- the 0/1/2 scale is a validated cardinal measurement scale;
- the nine dimensions are mutually independent;
- independent facilitators would elicit the same requirement profiles and salience weights;
- lower modelled debt predicts better DAO governance outcomes;
- the temporal debt metaphor is empirically calibrated.

The analysis is intentionally labelled as **author-conducted robustness and sensitivity analysis**, not validation. It tests internal model behaviour under explicit assumptions.

## Model definition

A decision `k` is represented over nine dimensions:

| Code | Dimension |
|---|---|
| A | acceptability |
| O | ordinal ordering |
| I | intensity |
| F | fallback transfer |
| M | pairwise majority |
| P | proportional-distributive structure |
| D | delegable agency |
| T | temporal persistence |
| J | justificatory objection |

For method `m` and dimension `d`, the capacity matrix gives `c_md` in `{0, 1, 2}`.

For a decision profile:

- `r_kd >= 0` is the requirement level;
- `r_kd = 0` marks dimension `d` inactive;
- `w_kd >= 0` is the salience weight;
- weights over active dimensions sum to `1`.

Default adequacy function:

```text
a_kdm = 1                                 if r_kd = 0
a_kdm = min(c_md, r_kd) / r_kd            if r_kd > 0
```

Debt vector:

```text
b_kd(m) = w_kd * (1 - a_kdm)
```

L1 issuance:

```text
I_k(m) = sum_d b_kd(m)
```

Repertoire floor:

```text
U_k = min_m I_k(m)
```

Avoidable excess:

```text
V_k(m_chosen) = I_k(m_chosen) - U_k
```

The implementation also supports sensitivity checks under `strict`, `sqrt_ratio`, and `presence_only` adequacy functions, and under `L1`, `L2`, and `max` scalarisation norms.

## Capacity matrix provenance

The capacity matrix is **author-specified** from a qualitative reading of each method's ballot/information format. It is not expert-calibrated and should not be treated as a settled measurement object.

This is why the repository includes per-entry influence and adequacy-function sensitivity checks. Researchers are encouraged to modify `data/capacity_matrix.csv`, rerun the script, and compare outputs.

## Scientific limits

The strongest defensible claim supported by this repository is:

> Given a stated capacity matrix, requirement profile, salience weights, adequacy function, and method repertoire, the Decision Debt computation is reproducible and can expose where modelled informational shortfall sits, which methods are frontier under those assumptions, and how sensitive those frontier sets are to modelling choices.

The repository does **not** support a claim that Decision Debt is an externally validated measure of governance quality.
